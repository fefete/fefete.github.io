VR map editor for Unity5

Summary

Final year project in Teesside University featuring an standalone application to modify and create maps in VR and then importing the changes to an Unity 5 Editor, for that, I used C# scripting and Oculus Rift utilities. The idea, is to be able to abstract the designers and artists to the engine itself and provide them with a more simple workflow while keeping the features of an inspector in a game engine. Featuring eye gaze control, this app is perfect for people that have Oculus rift, but does not have the extra pads for them

Techniques used and technical info

  • Unity 5
  • C#
  • Oculus Rift
  • Asset loading in runtime
  • Unity editor scripts
  • Friendly UI for VR
